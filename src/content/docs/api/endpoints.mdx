# API Endpoints

Sheety provides a full RESTful API for your Google Sheets data.

## Base URL

```
https://api.sheety.co/{username}/{project}/{sheet}
```

Replace:
- `{username}` - Your Sheety username
- `{project}` - Your project name
- `{sheet}` - The name of your sheet (tab)

## Available Endpoints

### GET - Read Data

Retrieve all rows or a specific row from your sheet.

#### Get All Rows

```http
GET /api/v1/{username}/{project}/{sheet}
```

**Example Request:**

```javascript
fetch('https://api.sheety.co/john/myproject/users')
  .then(response => response.json())
  .then(data => console.log(data));
```

**Example Response:**

```json
{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com",
      "age": 30
    },
    {
      "id": 2,
      "name": "Jane Smith",
      "email": "jane@example.com",
      "age": 25
    }
  ]
}
```

#### Get Single Row

```http
GET /api/v1/{username}/{project}/{sheet}/{id}
```

**Example Request:**

```javascript
fetch('https://api.sheety.co/john/myproject/users/2')
  .then(response => response.json())
  .then(data => console.log(data));
```

**Example Response:**

```json
{
  "user": {
    "id": 2,
    "name": "Jane Smith",
    "email": "jane@example.com",
    "age": 25
  }
}
```

### POST - Create Data

Add a new row to your sheet.

```http
POST /api/v1/{username}/{project}/{sheet}
```

**Example Request:**

```javascript
fetch('https://api.sheety.co/john/myproject/users', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    user: {
      name: "Bob Johnson",
      email: "bob@example.com",
      age: 35
    }
  })
})
```

**Example Response:**

```json
{
  "user": {
    "id": 3,
    "name": "Bob Johnson",
    "email": "bob@example.com",
    "age": 35
  }
}
```

### PUT - Update Data

Update an existing row in your sheet.

```http
PUT /api/v1/{username}/{project}/{sheet}/{id}
```

**Example Request:**

```javascript
fetch('https://api.sheety.co/john/myproject/users/2', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    user: {
      name: "Jane Doe",
      age: 26
    }
  })
})
```

**Example Response:**

```json
{
  "user": {
    "id": 2,
    "name": "Jane Doe",
    "email": "jane@example.com",
    "age": 26
  }
}
```

### DELETE - Remove Data

Delete a row from your sheet.

```http
DELETE /api/v1/{username}/{project}/{sheet}/{id}
```

**Example Request:**

```javascript
fetch('https://api.sheety.co/john/myproject/users/3', {
  method: 'DELETE'
})
```

**Example Response:**

```json
{
  "success": true
}
```

## Query Parameters

### Filtering

Filter results using query parameters:

```http
GET /api/v1/{username}/{project}/{sheet}?filter[column]=value
```

**Example:**

```javascript
// Get all users with age 30
fetch('https://api.sheety.co/john/myproject/users?filter[age]=30')
```

### Sorting

Sort results by a column:

```http
GET /api/v1/{username}/{project}/{sheet}?sort=column&order=asc
```

**Example:**

```javascript
// Sort users by age in descending order
fetch('https://api.sheety.co/john/myproject/users?sort=age&order=desc')
```

### Pagination

Limit and offset results:

```http
GET /api/v1/{username}/{project}/{sheet}?limit=10&offset=20
```

**Example:**

```javascript
// Get 10 users, starting from the 21st row
fetch('https://api.sheety.co/john/myproject/users?limit=10&offset=20')
```

## Response Format

All responses are in JSON format. The data is wrapped in an object with the sheet name as the key.

### Success Response

```json
{
  "sheetName": [
    { ... },
    { ... }
  ]
}
```

### Error Response

```json
{
  "error": "Error message",
  "status": 400
}
```

## Status Codes

| Code | Description |
|------|-------------|
| 200 | Success |
| 201 | Created |
| 400 | Bad Request - Invalid data format |
| 401 | Unauthorized - Authentication required |
| 404 | Not Found - Resource doesn't exist |
| 429 | Too Many Requests - Rate limit exceeded |
| 500 | Internal Server Error |

## Rate Limits

- **Free Plan**: 200 requests per hour
- **Pro Plan**: 1000 requests per hour
- **Enterprise**: Custom limits

Rate limit headers are included in every response:

```
X-RateLimit-Limit: 200
X-RateLimit-Remaining: 195
X-RateLimit-Reset: 1640000000
```
