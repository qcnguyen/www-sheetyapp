## Why Sheety?

Sheety makes it incredibly easy to use Google Sheets as a backend for your applications. Whether you're building a prototype, managing content, or creating a simple database, Sheety provides a straightforward API interface to your spreadsheet data.

## Getting Started

Ready to get started? Check out our [Quick Start Guide](/docs/quick-start/) to create your first API in minutes.

## Features

- **Easy Setup**: Connect your Google Sheet and get an instant API
- **RESTful API**: Standard HTTP methods (GET, POST, PUT, DELETE)
- **Authentication**: Secure your endpoints with API keys
- **Real-time Updates**: Changes to your sheet are immediately reflected in the API
- **No Backend Required**: Use Google Sheets as your database
