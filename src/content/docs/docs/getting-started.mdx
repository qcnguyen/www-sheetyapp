## Your First Workbook in 3 Steps

### Step 1: Create a Workbook

1. Navigate to your Sheety App instance
2. Click **"Create Workbook"** from the sidebar
3. Enter a name and description
4. Click **"Save"**

### Step 2: Add a Sheet

1. From your workbook home, click **"CREATE A SHEET"**
2. Enter a sheet name (e.g., "Employees", "Projects", "Inventory")
3. Add a description
4. Click **"Save"**

### Step 3: Build Your Data Structure

1. Right-click anywhere in the empty sheet
2. Select **"Add Column"**
3. Define column properties (name, type, validation)
4. Repeat for all needed columns

## Key Features

### Workbooks
Workbooks are containers for your sheets and apps. Think of them as projects or databases - each workbook can contain multiple sheets and custom applications.

### Sheets
Sheets are where your data lives. Similar to database tables or spreadsheet tabs, sheets have:
- **Columns** - Define what type of data you store
- **Rows** - Individual records of data
- **Views** - Different ways to visualize and filter your data

### Apps
Build custom applications on top of your data without coding. Create forms, dashboards, and interfaces that interact with your sheets.

## Navigation Overview

The Sheety App interface includes:

- **Left Sidebar** - Quick access to workbooks and settings
- **Top Bar** - Context-specific actions and notifications
- **Workbook Sidebar** - Navigate between sheets, apps, and settings within a workbook
- **Main Content Area** - Where you work with your data
